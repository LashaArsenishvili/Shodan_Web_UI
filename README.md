# Shodan Explorer Pro

**შოდანი მკვლევარი პრო**  
**ქართული / ინგლისური ბილინგვალური ინტერფეისი**  
მოწინავე ვებ-დაშბორდი Shodan.io-სთვის

### რას აკეთებს აპლიკაცია?

**Shodan Explorer Pro** არის სრულფასოვანი, თანამედროვე და მოსახერხებელი ვებ-ინტერფეისი Shodan-ის API-სთვის.  
იგი საშუალებას გაძლევთ:

- მარტივად და სწრაფად მოძებნოთ მილიარდობით ინტერნეტ-მოწყობილობა (სერვერები, IoT, ICS, კამერები, როუტერები და ა.შ.)
- გამოიყენოთ როგორც მარტივი, ისე უაღრესად დეტალური ფილტრები
- ნახოთ ვიზუალიზაციები (ჩარტები, რუკა)
- გააკეთოთ ერთ IP-ის სიღრმისეული ანალიზი + სრული ისტორია
- გამოიყენოთ **ქართული OSINT ბაზა** (სახელი + გვარი ან პირადი ნომერი)

აპლიკაცია **ავტომატურად იწყება ქართულად (KA)** და ერთი ღილაკით გადადის ინგლისურზე (EN).

|---------------------------|--------|

| **API Key მართვა**         | ავტომატური ჩატვირთვა `api.txt`-დან + შემოწმება |

| **ჩარტები**               | ტოპ ქვეყნები, პორტები, ორგანიზაციები, პროდუქტები, OS |

| **ინტერაქტიული რუკა**     | ყველა გეო-ლოკაციური ჰოსტი Plotly Geo-ზე |

| **OSINT მოდული**           | ქართული მონაცემთა ბაზების ძებნა (სახელი+გვარი ან პირადი ნომერი) |
| **თემა**                   | მუქი კიბერ-პანკ სტილი (შავი + ცისფერი აქცენტები) |

### ფილტრების ჯგუფები (Sidebar)

- **🌐 ქსელი** (IP, Port, Net, ASN, ISP)
- **🏳️ მდებარეობა** (Country, City, Region, Geo, Postal)
- **🖥️ სისტემა** (OS, Hostname, Domain, Org)
- **🔧 პროგრამა** (Product, Version, CPE, HTTP title, hash-ები)
- **🔐 SSL / TLS** (სერტიფიკატის ყველა პარამეტრი)
- **⚙️ სერვისები** (Tags, Category, Cloud, Device, Screenshot, Vuln)
- **📡 პროტოკოლები** (SSH, FTP, MongoDB, RDP, Telnet, SNMP და ა.შ.)

### ტექნოლოგიები

- **Python 3.12** + **Dash** (Plotly)
- **Dash Bootstrap Components** + მორგებული CSS (JetBrains Mono + Noto Sans Georgian)
- **Plotly** — ჩარტები და ინტერაქტიული რუკა
- **Pandas** — მონაცემების დამუშავება
- **Requests** — Shodan API
- **local OSINT** — საკუთარი `.txt` ბაზები (tab-separated)
